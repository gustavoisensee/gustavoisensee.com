Scenario Type
Basic Path Basic Path
Details:
Steps:
1. Administrador clica no ícone Escolas no meu lateral
2. Sistema abre a tela de Escolas
3. Sistema carrega uma listagem de escolas. UC002.1 - Listar Escolas
4. Sistema possibilita o Administrador inserir escola. UC002.2 - Inserir Escola
5. Sistema possibilita o Administrador editar escola. UC002.3 - Editar Escola
6. Sistema possibilita o Administrador excluir escola. UC002.4 - Excluir Escola
Novo Alternate
Details:
Steps:
1. Sistema abre tela modal de cadastro de escola
2. Sistema carrega campos do tipo listas
3. Administrador preenche os campos e clica em salvar
4. Sistema valída campos obrigatorios
5. Se valído: Sistema grava os dados, fecha a tela e atualiza a tabela da tela anterior
6. Se inválido: Sistema informa o motivo da invalidez.
Alterar Alternate
Details:
Steps:
1. Sistema abre tela modal de alteração da escola
2. Sistema carrega campos
3. Administrador preenche os campos e clica em salvar
4. Sistema valída campos obrigatorios
5. Se valído: Sistema grava os dados, fecha a tela e atualiza a tabela da tela anterior
6. Se inválido: Sistema informa o motivo da invalidez.
Excluir Alternate
Details:
Steps:
1. Sistema abre uma modal de confirmação de exclusão da escola
2. Administrador clica no botão 'Sim' para excluir a escola selecionada
3. Sistema valída se é possível excluir a escola
4. Se possível: Sistema excluir a escola, fecha a modal de confirmação e atualiza a tabela da página anterior
5. Se não for possível: Sistema informa ao Administrador que não pode ser excluído e fecha a modal de confirmação
6. Administrador clica no botão 'Não'
7. Sistema fecha a modal de confirmação e não excluir a escola
