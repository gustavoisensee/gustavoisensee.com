Scenario Type
Basic Path Basic Path
Details:
Steps:
1. Sistema exibe todos os campos desabilitados exceto Ano, Escola, Série e a Tabela
2. O Aluno ou Responsável seleciona Ano, Escola, Aluno e Serie
3. Sistema habilita e carrega no combobox aluno os alunos referentes aos filtros selecionados e caso o perfil seja aluno no combo terá apenas seu próprio registro caso o perfil do usuário seja Responsável o combo é carrega com todos os alunos os quais o Responsável possui vínculo
4. Sistema habilita e carrega no combobox turma as turmas referentes aos filtros selecionados
5. O Aluno ou Responsável clica no botão Pesquisar
6. Sistema busca uma nova lista de materiais referente aos filtros selecionados
7. Sistema exibe na tabela a nova lista
