O Administrador deve estar com a tela modal de nova escola ou editar escola e ter aberto a aba de professores. Functional Proposed Medium Medium Moderate
Details:
 
Scenario Type
Basic Path Basic Path
Details:
Steps:
1. O Administrador clica no botão Remover em uma linha na tabela exibida
2. Sistema abre modal de confirmação de exclusão. SC002.5 - Remover vínculo do professor com a escola
3. Sistema possibilita o Administrador clicar em Sim ou Não
4. O Administrador clica em Sim
5. Sistema valida se é possível desvincular o professor da escola.
6. Sistema remove o vínculo do professor com a escola
7. Sistema atualiza a listagem de professores vínculados a escola

3a. Não (at End)
