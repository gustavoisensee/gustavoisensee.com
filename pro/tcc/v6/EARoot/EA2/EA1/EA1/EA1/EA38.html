Scenario Type
Basic Path Basic Path
Details:
Steps:
1. O usuário acessa o sistema
2. Sistema abre a tela de login SC001 - Login
3. O usuário informa o email e senha e clica em logar
4. Sistema valída a autenticidade do usuário e senha informados
5. Sistema redireciona o para a tela principal.
6. Sistema habilita os menus de acordo com o perfil do Usuário logado
Exceptions:
4a. Usuário e senha incorretos (at End)
Usuário e senha corretos Alternate
Details:
