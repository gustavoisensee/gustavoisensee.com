Scenario Type
Basic Path Basic Path
Details:
Steps:
1. O usuário acessa o sistema
2. Sistema abre a tela de login
3. O usuário informa o email e senha e clica em logar
4. Sistema valída a autenticidade do usuário e senha informados
Exceptions:
4a. Usuário e senha corretos (at End)
4b. Usuário e senha incorretos (at End)
Usuário e senha corretos Alternate
Details:
Steps:
1. Sistema redireciona o para a tela principal.
2. Sistema habilita os menus de acordo com o perfil do Usuário logado
Usuário e senha incorretos Alternate
Details:
Steps:
1. Sistema informa através de uma mensagem ' Usuário ou senha incorretos' e limpa os dados de acesso.
